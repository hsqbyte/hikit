# 待办事项（Todo）

> 资产树新增"待办事项"类型，轻量级任务管理

## 🔴 高优先级 — 核心功能

- [ ] **待办事项资产类型** — 资产树新增 `todo` 类型，支持新建/删除待办清单
- [ ] **任务列表 UI** — 双击待办资产打开任务列表视图，展示所有任务项
- [ ] **新增任务** — 输入框快速添加新任务，支持回车提交
- [ ] **完成/取消完成** — 点击复选框切换任务状态（未完成 / 已完成）
- [ ] **删除任务** — 滑动删除 / 右键删除 / 批量删除
- [ ] **任务编辑** — 双击任务项编辑标题内容
- [ ] **数据持久化** — 待办数据存储到 SQLite（todo_items 表：id, list_id, title, completed, sort_order, created_at, updated_at）

## 🟡 中优先级 — 增强功能

- [ ] **优先级标记** — 支持设置任务优先级（高 🔴 / 中 🟡 / 低 🟢 / 无）
- [ ] **截止日期** — 为任务设置到期时间，过期任务高亮提醒
- [ ] **分类/标签** — 任务支持添加标签（颜色标签），按标签筛选
- [ ] **拖拽排序** — 拖动任务项调整顺序
- [ ] **子任务** — 支持任务嵌套，展开/折叠子任务列表，子任务进度条
- [ ] **任务筛选** — 按状态筛选：全部 / 未完成 / 已完成 / 已过期
- [ ] **批量操作** — 全选 / 批量完成 / 批量删除
- [ ] **任务计数** — 资产树节点旁显示未完成任务数角标

## 🟢 低优先级 — 高级功能

- [ ] **任务备注** — 展开任务详情，添加文本备注
- [ ] **提醒通知** — 截止日期到达时系统通知提醒
- [ ] **重复任务** — 支持设置每日/每周/每月重复
- [ ] **任务统计** — 完成率、趋势图表
- [ ] **任务搜索** — 全局搜索任务标题和备注内容
- [ ] **导出** — 导出任务列表为 Markdown / CSV
- [ ] **回收站** — 已删除任务可恢复
