# 密码保险箱（Vault）

> 资产树新增"密码保险箱"类型，安全存储账户密码、API Key 等敏感信息

## 🔴 高优先级 — 核心功能

- [ ] **密码保险箱资产类型** — 资产树新增 `vault` 类型，支持新建/删除保险箱
- [ ] **凭证列表 UI** — 双击保险箱资产打开凭证列表，卡片/列表布局展示所有条目
- [ ] **新增凭证** — 添加凭证条目（标题、用户名、密码、URL、备注）
- [ ] **密码显示/隐藏** — 密码默认隐藏（`••••••`），点击眼睛图标切换显示
- [ ] **一键复制** — 点击复制按钮快速复制用户名/密码/URL 到剪贴板
- [ ] **编辑凭证** — 双击凭证条目编辑各字段
- [ ] **删除凭证** — 右键删除 / 批量删除
- [ ] **数据持久化** — 凭证存储到 SQLite（vault_items 表：id, vault_id, title, username, password, url, notes, category, sort_order, created_at, updated_at）

## 🟡 中优先级 — 增强功能

- [ ] **密码加密** — 使用 AES-256 加密存储密码字段，主密码解锁
- [ ] **搜索过滤** — 按标题、用户名、URL 搜索凭证
- [ ] **分类管理** — 凭证支持分类（网站、服务器、API、银行卡、其他），按分类筛选
- [ ] **密码生成器** — 内置随机密码生成（可配置长度、字符类型）
- [ ] **密码强度指示** — 显示密码强度条（弱/中/强）
- [ ] **收藏/置顶** — 常用凭证置顶显示
- [ ] **自定义字段** — 凭证支持添加自定义键值对（如安全问题、PIN 码等）

## 🟢 低优先级 — 高级功能

- [ ] **主密码保护** — 打开保险箱时需输入主密码验证
- [ ] **自动锁定** — 空闲一段时间后自动锁定保险箱
- [ ] **导入/导出** — 支持从 CSV / JSON 导入导出凭证
- [ ] **密码过期提醒** — 设置密码有效期，过期自动提醒更换
- [ ] **历史记录** — 记录密码修改历史
- [ ] **回收站** — 已删除凭证可恢复
- [ ] **OTP/TOTP** — 支持存储和生成两步验证码
