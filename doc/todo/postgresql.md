# PostgreSQL 管理模块

> 参考 HexHub / MexHub 风格的数据库管理 UI

## 🔴 高优先级 — 连接 & 浏览

- [x] **连接管理** — 资产树新增 PostgreSQL 连接类型，配置 Host / Port / User / Password / Database / SSL
- [x] **连接测试** — 创建/编辑连接时一键测试连通性
- [x] **数据库列表** — 连接成功后展示服务器上所有数据库（左侧树节点）
- [x] **Schema 浏览** — 展开数据库后列出 `public` 等 Schema，树结构：连接 → 数据库 → Schema → 分类节点
- [x] **表列表** — Schema 下以图标网格 / 列表展示所有表（参考截图第一张的网格布局）
- [x] **表数据浏览** — 点击表名打开数据网格（DataGrid），分页加载数据，支持列排序
- [x] **列筛选 / 过滤** — 每列头部支持 筛选图标（`▽`），输入条件过滤行

## 🟡 中优先级 — 数据操作 & 查询

- [x] **新增行** — 数据网格中点击「新建」按钮插入新行（参考截图中的 `+` 按钮）
- [x] **编辑行** — 双击单元格直接编辑，修改后高亮标记，手动提交/回滚
- [x] **删除行** — 选中行后点击删除，或右键删除
- [x] **SQL 查询编辑器** — 内置 SQL 编辑器（语法高亮 + 自动补全），执行查询并在下方展示结果
- [x] **多 Tab 管理** — 同时打开多张表 / 多个查询，Tab 栏切换（参考截图顶部 Tab 栏）
- [x] **DDL 查看** — 一键查看表的 `CREATE TABLE` DDL 语句（参考截图右上角 DDL 按钮）
- [x] **表结构查看** — 查看列名、类型、默认值、约束、索引等元数据

## 🟡 中优先级 — 对象管理

- [x] **视图（View）列表** — 展示并浏览视图
- [x] **函数 / 存储过程** — 列出数据库函数，支持查看/编辑函数体
- [x] **物化视图** — 列出物化视图，支持手动刷新
- [ ] **新建表** — 图形化建表向导（列定义、主键、约束、索引）
- [ ] **修改表结构** — 图形化 ALTER TABLE（添加/删除/修改列）
- [x] **删除表** — 右键删除表（带确认对话框）

## 🟢 低优先级 — 高级功能

- [ ] **数据导出** — 导出查询结果 / 表数据为 CSV / JSON / SQL INSERT
- [ ] **数据导入** — 从 CSV / SQL 文件导入数据到表
- [ ] **查询历史** — 记录已执行的 SQL，支持搜索和重放
- [ ] **收藏查询** — 常用 SQL 语句收藏管理
- [ ] **表关系图（ER 图）** — 根据外键自动生成表关系可视化
- [ ] **数据库统计** — 显示表大小、行数、索引使用情况等
- [ ] **用户权限管理** — 查看/管理数据库用户和权限
