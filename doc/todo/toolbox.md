# 开发者工具箱（Toolbox）

> Activity Bar 新增「工具箱」入口（🔧 图标），点击后左侧展示工具分类列表，右侧 Tab 打开对应工具面板。
> 工具为无状态面板，无需资产管理。

## 🔴 高优先级 — 框架 & 高频工具

- [x] **Activity Bar 入口** — 新增工具箱图标，点击切换到工具列表视图
- [x] **工具列表 UI** — 左侧展示所有工具分类列表（图标 + 名称），点击在右侧 Tab 打开
- [x] **JSON 格式化** — 粘贴 JSON 一键格式化 / 压缩 / 校验，支持 JSON ↔ YAML 互转
- [x] **编码 / 解码** — Base64 编解码、URL Encode/Decode、HTML Entity、Unicode 转换
- [x] **时间戳转换** — Unix 时间戳 ↔ 可读日期互转，支持毫秒/秒切换，时区选择
- [x] **Hash 计算** — MD5 / SHA1 / SHA256 / SHA512，支持文本输入和文件拖入计算

## 🟡 中优先级 — 开发辅助

- [x] **正则表达式测试器** — 输入正则 + 测试文本，实时高亮匹配结果，显示分组捕获
- [x] **Diff 对比** — 左右双栏文本对比，高亮差异行和字符级差异
- [x] **JWT 解析** — 粘贴 JWT Token 自动解码 Header / Payload，显示过期时间
- [x] **UUID 生成器** — 一键生成 UUID v4，支持批量生成和格式选择（大小写/带横杠）
- [x] **Cron 表达式** — 输入 cron 表达式，解析含义 + 预览未来 N 次执行时间，支持可视化编辑生成
- [x] **颜色工具** — 取色器 + HEX / RGB / HSL 格式互转 + 调色板

## 🟢 低优先级 — 扩展工具

- [x] **IP / 网络查询** — IP 归属地查询、子网计算器、本机网络信息
- [x] **二维码生成** — 输入文本/URL 生成二维码图片，支持下载
- [x] **图片工具** — 图片压缩、格式转换（PNG/JPG/WebP）、Base64 互转
- [x] **文本处理** — 大小写转换、去重、排序、行号添加、空行清理等批量文本操作
- [x] **Markdown 预览** — 独立的 Markdown 实时预览工具（可从备忘录复用组件）
- [x] **端口扫描** — 检查本机/远程指定端口范围的开放状态
