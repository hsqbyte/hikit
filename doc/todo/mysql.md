# MySQL 管理模块

> 参考 PostgreSQL 模块风格，适配 MySQL 特性

## 🔴 高优先级 — 连接 & 浏览

- [ ] **连接管理** — 资产树新增 MySQL 连接类型，配置 Host / Port / User / Password / Database / SSL
- [ ] **连接测试** — 创建/编辑连接时一键测试连通性
- [ ] **数据库列表** — 连接成功后展示服务器上所有数据库（左侧树节点）
- [ ] **表列表** — 数据库下以网格 / 列表展示所有表
- [ ] **表数据浏览** — 点击表名打开数据网格（DataGrid），分页加载数据，支持列排序
- [ ] **列筛选 / 过滤** — 每列支持条件过滤

## 🟡 中优先级 — 数据操作 & 查询

- [ ] **新增行** — 数据网格中插入新行
- [ ] **编辑行** — 双击单元格直接编辑，修改后高亮标记，手动提交/回滚
- [ ] **删除行** — 选中行后删除
- [ ] **SQL 查询编辑器** — 内置 SQL 编辑器（语法高亮 + 自动补全），执行查询并展示结果
- [ ] **多 Tab 管理** — 同时打开多张表 / 多个查询
- [ ] **DDL 查看** — 一键查看 `SHOW CREATE TABLE` DDL 语句
- [ ] **表结构查看** — 查看列名、类型、默认值、约束、索引等元数据

## 🟡 中优先级 — 对象管理

- [ ] **视图（View）列表** — 展示并浏览视图
- [ ] **函数 / 存储过程** — 列出函数和存储过程，支持查看/编辑
- [ ] **触发器** — 列出触发器，支持查看/编辑
- [ ] **新建表** — 图形化建表向导（列定义、主键、约束、索引、引擎选择）
- [ ] **修改表结构** — 图形化 ALTER TABLE（添加/删除/修改列）
- [ ] **删除表** — 右键删除表（带确认对话框）

## 🟢 低优先级 — 高级功能

- [ ] **数据导出** — 导出查询结果 / 表数据为 CSV / JSON / SQL INSERT
- [ ] **数据导入** — 从 CSV / SQL 文件导入数据到表
- [ ] **查询历史** — 记录已执行的 SQL，支持搜索和重放
- [ ] **收藏查询** — 常用 SQL 语句收藏管理
- [ ] **表关系图（ER 图）** — 根据外键自动生成表关系可视化
- [ ] **数据库统计** — 显示表大小、行数、索引使用情况等
- [ ] **用户权限管理** — 查看/管理数据库用户和权限
- [ ] **字符集管理** — 查看/设置数据库和表的字符集与排序规则
