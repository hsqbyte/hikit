# 备忘录（Memo）

> 资产树新增"备忘录"类型，轻量级笔记管理

## 🔴 高优先级 — 核心功能

- [ ] **备忘录资产类型** — 资产树新增 `memo` 类型，支持新建/删除备忘录
- [ ] **备忘录编辑器** — 双击备忘录资产打开富文本 / Markdown 编辑器
- [ ] **Markdown 支持** — 支持 Markdown 语法编写，实时预览（编辑/预览/分屏三种模式）
- [ ] **自动保存** — 编辑内容自动保存（防抖策略，停止输入 1s 后自动保存）
- [ ] **数据持久化** — 备忘录内容存储到 SQLite（memos 表：id, asset_id, title, content, content_type, created_at, updated_at）

## 🟡 中优先级 — 增强功能

- [ ] **富文本编辑** — 支持加粗、斜体、列表、代码块、链接、图片等格式
- [ ] **代码高亮** — 备忘录中的代码块支持语法高亮（多语言）
- [ ] **字数统计** — 编辑器底部实时显示字数 / 字符数 / 行数
- [ ] **全文搜索** — 搜索所有备忘录内容，关键词高亮
- [ ] **快捷键** — 常用编辑操作快捷键（Ctrl+B 加粗、Ctrl+S 保存等）
- [ ] **标签/分类** — 备忘录支持添加标签，按标签筛选
- [ ] **置顶/收藏** — 重要备忘录置顶或标记收藏
- [ ] **修改历史** — 记录备忘录修改版本，支持查看历史 / 回滚

## 🟢 低优先级 — 高级功能

- [ ] **附件支持** — 备忘录中嵌入 / 附加文件（图片、PDF 等）
- [ ] **模板** — 预设备忘录模板（会议记录、日报、技术笔记等）
- [ ] **导出** — 导出为 Markdown / PDF / HTML
- [ ] **备忘录链接** — 备忘录之间互相引用链接
- [ ] **字体/主题** — 编辑器字体大小、行间距、主题配色自定义
- [ ] **回收站** — 已删除备忘录可恢复
