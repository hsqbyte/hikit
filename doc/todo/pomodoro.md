# 🍅 番茄钟 (Pomodoro Timer)

## 概述
集成番茄工作法计时器，帮助开发者科学管理工作和休息时间。

## 功能清单

### Phase 1 - 基础计时器
- [ ] Activity Bar 添加番茄钟图标 (🍅)
- [ ] PomodoroPanel 侧边面板组件
  - [ ] 开始/暂停/停止按钮
  - [ ] 圆环倒计时动画（SVG 环形进度条）
  - [ ] 显示当前状态：工作中 / 短休息 / 长休息
  - [ ] 显示当前是第几个番茄
- [ ] PomodoroView 主内容区（大号显示）
  - [ ] 全屏倒计时视图
  - [ ] 精美的渐变动画效果
- [ ] 计时逻辑
  - [ ] 默认：25分钟工作 + 5分钟短休息
  - [ ] 每4个番茄后长休息 15分钟
  - [ ] 自动切换工作/休息模式

### Phase 2 - 通知与音效
- [ ] macOS 桌面通知（时间到）
- [ ] 可选提示音（滴答声、铃声）
- [ ] 可选白噪音背景（雨声、咖啡馆）

### Phase 3 - 统计与持久化
- [ ] 今日统计：完成番茄数、总专注时长
- [ ] 历史记录：每日/每周番茄数图表
- [ ] 数据持久化到本地 JSON

### Phase 4 - 高级功能
- [ ] 自定义时长设置（工作/短休/长休）
- [ ] 番茄关联 Todo 任务（专注在哪个任务上）
- [ ] 每日目标设定（目标 X 个番茄）

## 技术方案
- 前端纯 React 实现，不需要 Go 后端（计时用 setInterval）
- 通知使用 Wails runtime 的 Notification API
- 统计数据存 localStorage 或 Go 后端 JSON 文件
- 圆环动画用 SVG `stroke-dashoffset` 实现
