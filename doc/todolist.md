# FastTool TODO List

## SSH / SFTP 模块

### 🔴 高优先级

- [x] **文件上传** — 点击上传按钮选择本地文件上传到远程（后端 `UploadFile` 已实现）
- [x] **文件下载** — 点击下载按钮保存远程文件到本地（后端 `ReadFile` 已实现）
- [ ] **拖拽上传** — 从桌面/Finder 拖拽文件到文件管理器自动上传
- [ ] **拖拽下载** — 从文件管理器拖拽文件到桌面/Finder 自动下载
- [x] **文件重命名** — 前端接入已有的 `SFTPRename` 绑定（双击名称 / 右键重命名）
- [x] **右键文件菜单** — 下载 / 重命名 / 删除 / 复制路径 / 查看属性

### 🟡 中优先级

- [x] **文件编辑/查看** — 双击小文件在内置编辑器中查看/编辑（后端 `ReadFile`/`WriteFile` 已有）
- [x] **SSH 重连** — 连接断开后显示"重新连接"按钮
- [ ] **多标签终端** — 同一 SSH 连接支持打开多个终端 tab
- [ ] **文件搜索** — 支持远程 `find` 搜索，不仅仅是本地过滤
- [ ] **SSH 密钥 passphrase** — 支持有密码保护的私钥

### 🟢 低优先级

- [ ] **文件权限修改** — `chmod` 操作
- [ ] **文件复制/移动** — 远程服务器上复制/移动文件
- [ ] **传输队列** — 批量上传/下载时显示传输进度列表
- [ ] **隐藏文件切换** — 显示/隐藏 `.` 开头的文件
