## SSH 端口转发（代理）

> 参考 Termius Port Forwarding 功能

### 🔴 高优先级

- [ ] **本地转发 (Local Forwarding)** — 将远程服务器端口映射到本地（`ssh -L`），访问远程数据库、内网服务等
- [ ] **远程转发 (Remote Forwarding)** — 将本地端口暴露到远程服务器（`ssh -R`），内网穿透
- [ ] **动态转发 (Dynamic / SOCKS)** — 建立 SOCKS5 代理（`ssh -D`），通过 SSH 隧道代理所有流量

### 🟡 中优先级

- [ ] **转发规则管理** — 新建 / 编辑 / 删除转发规则（名称、类型、本地端口、远程地址:端口、关联 SSH 连接）
- [ ] **转发规则列表 UI** — 卡片/网格布局展示所有转发规则，显示状态（运行中/已停止）
- [ ] **一键启停** — 点击卡片启动/停止转发，支持批量操作
- [ ] **状态指示** — 实时显示转发连接状态（已连接/断开/错误）

### 🟢 低优先级

- [ ] **转发规则持久化** — 规则保存到 SQLite，重启后保留
- [ ] **自动重连** — 转发断开后自动尝试重连
- [ ] **流量统计** — 显示每条转发规则的流量使用情况
